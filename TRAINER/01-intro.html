01-intro

        Hinter einem HTML-Tag steht immer ein HTML Element (ein JS Objekt)
        CSS-Stile sind bei diesen HTML Element in der Eigenschaft 'style' zusammengefasst.
        Schauen Sie beim Objekt window im der Konsole nach mehr Infos nach.
        Beim globalen Objekt sind auch die Konstruktoren von HTML Element und viele weitere Objekte gesammelt.
