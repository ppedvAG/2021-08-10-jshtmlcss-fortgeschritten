06-css-inhertinece

Von wo kommen die CSS-Stile für Elemente?
    - vom Browser (defaults). Wie berechnet der Browser die Stile?
        - manche vererben die Stile von Elternelementen
        - manche Stile sind relativ default-Werten von anderen Elementen (z.B. je größer die Schrift bei html, desto größer ist sie bei h1)
    - vom Entwickler (diese sind stärker als die vom browser). Wie werden diese weiter berechnet?
        - anhand von Spezifität 
        - anhand von Reihenfolge von CSS-Regeln
Kindelemente können die CSS-Stile von ihren Elternelementen vererben

z.B. bei Eigenschaft font-size haben viele Elemente default-Wert bei 16px;
Dieser Default wird aber anhand von einem anderen berechnet und nämlich 'vererbe vom Elternelement' (inherit);
Diese Vererbung kann man zurück zum 'initial'-Wert zurücksetzen

Werte für Prop border werden per default nicht vererbt.
Man kann aber die Kindelemente dazu zwingen - Wert 'inherit'

Sehr hilfreich sind die DevTools beim Browser, Bereich 'Elements'.
